import { IconCheckbox } from "@tabler/icons-react"

| Entity          | Behavior                                                                                                                              | Requires Redstone? |
| --------------- | --------------------------------------------------------------------------------------------------------------------------------------| ------------------ |
| Allays          | Vacuums up nearby items[^1]                                                                                                           |                    |
| Armadillos      | Periodically produces Armadillo Scutes[^1]                                                                                            |                    |
| Blazes          | Emits light                                                                                                                           |                    |
| Breezes         | Fires a Breeze Wind Charge                                                                                                            | <IconCheckbox />   |
| Cats            | Produces a gift when its owner sleeps nearby. Also emits a redstone signal when owner is online[^1]                                   |                    |
| Chickens        | Periodically produces Eggs[^1]                                                                                                        |                    |
| Creepers        | Plays Creeper fuse sound                                                                                                              | <IconCheckbox />   |
| Decoys          | Pulls all nearby entities toward the jar                                                                                              | <IconCheckbox />   |
| Elder Guardians | Inflicts Mining Fatigue III onto all nearby players                                                                                   | <IconCheckbox />   |
| Ender Dragons   | Using a Bottle on the jar produces Dragon's Breath                                                                                    |                    |
| Frogs           | Consumes nearby Magma Cubes to produce Froglights[^1]                                                                                 |                    |
| Glow Squids     | Emits light                                                                                                                           |                    |
| Mooshrooms      | Using a Bucket or Bowl on the jar produces Milk Buckets and Mushroom Stew respectively. Shearing turns them into Cows                 |                    |
| Pandas          | Periodically produces Slimeballs[^1]                                                                                                  |                    |
| Piglins         | Vacuums up nearby Gold Ingots and produces Bartering loot[^1]                                                                         |                    |
| Pufferfish      | Produces a redstone signal when a player is nearby                                                                                    |                    |
| Sheep           | Can be sheared, periodically eating grass below to regrow their wool[^1]                                                              |                    |
| Sniffers        | Periodically generates Sniffer loot[^1]                                                                                               |                    |
| Snow Golems     | Fires a Snowball                                                                                                                      | <IconCheckbox />   |
| Villagers       | Can be traded with and will restock as usual                                                                                          |                    |
| Withers         | Breaks blocks in a 3x3x3 volume centered on the jar                                                                                   | <IconCheckbox />   |
| Default         | Right-click interactions (including with items) are carried through. Plays the mob's idle sound when powered                          | <IconCheckbox />   |


[^1]: Items will attempt to be deposited into directly adjacent inventories.

| Creepers        | Plays the Creeper fuse sound                                                                                                                                               | <IconCheckbox />   |
| Decoys          | Pulls toward the jar all nearby entities within a 21x21x21 area, centered on the jar                                                                                       | <IconCheckbox />   |

| Sniffers        | Periodically generates Sniffer loot[^1]                                                                                                                                    |                    |
| Snow Golems     | Fires a Snowball                                                                                                                                                           | <IconCheckbox />   |
| Villagers       | Can be traded with and will restock as usual                                                                                                                               |                    |
| Withers         | Breaks blocks in a 3x3x3 volume centered on the jar                                                                                                                        | <IconCheckbox />   |
| Default         | Right-click interactions (including with items) are carried through. Plays the mob's idle sound when powered                                                               | <IconCheckbox />   |

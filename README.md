# Ars Auxilia

Out-of-game libraries and apps for [Ars Nouveau](https://www.curseforge.com/minecraft/mc-mods/ars-nouveau).

## Packages

| Package | Description |
|---|---|
| [apps/ars.guide](apps/ars.guide) | Documentation Site ([Ars.Guide](https://ars.guide)) |
| [packages/addon-builder](packages/addon-builder) | Utilities for interacting with [ArsAddonBuilder](https://github.com/Jarva/ArsAddonBuilder) |

## Prerequisites

- [Node.js](https://nodejs.org) (LTS)
- [pnpm](https://pnpm.io) 10.17+
